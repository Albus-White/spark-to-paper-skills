repeated. Useful for diagnosing recurring AI-draft problems and infeasible experiment setups.

## Entry format

```
Case ID:
Source paper:
What was attempted:
Why it failed / blocker:
What was missing (data/code/compute/info):
Lesson / candidate rule (if any):
```
